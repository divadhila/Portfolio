Beef Price Prediction

A web application that predicts beef prices in Indonesia using a Recurrent Neural Network (RNN) model trained on Bapanas data, helping users track trends and make informed decisions.

What is the project about?

The Beef Price Prediction Web Application is a data-driven project designed to help users forecast beef prices in Indonesia using advanced machine learning techniques. The application leverages a Recurrent Neural Network (RNN) model, which was trained on historical beef price datasets obtained from Badan Pangan Nasional (Bapanas). By analyzing time-series data, the system is able to capture patterns and trends to generate accurate price predictions.

The frontend development aspect of this project ensures that the prediction results are presented in a clear and user-friendly interface. Users can easily input data, run predictions, and view outcomes directly on the website, making it practical for both researchers and the general public interested in market price forecasting.
