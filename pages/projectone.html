Sensor-Based Push-up and Sit-up Counter System

A sensor-based system developed with MPU sensors and BLE communication to monitor push-up and sit-up movements. The results are displayed on a web interface, allowing users to track performance, accuracy, and history of their exercises.

What is the project about?

The Push-up and Sit-up Counter System was developed by a team of three members using MPU sensors and Bluetooth Low Energy (BLE) communication. The system connects to an ESP32 server, detects correct and incorrect movements, provides real-time feedback, and stores data in CSV format. The processed results are then displayed on a web interface, allowing users to monitor their performance in a clear and interactive way.

The web development part was built using HTML, CSS, and JavaScript to design a user-friendly interface where users can view workout results directly on the website.
