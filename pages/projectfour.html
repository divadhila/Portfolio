Shopping List Web Application

A simple and interactive shopping list app built with Vite + React.js. This project helps users create, manage, and organize their shopping items efficiently. Items can be marked as purchased, sorted, and cleared with just a few clicks, making grocery planning easier and more organized.

What is the project about?

The Shopping List App was developed as a lightweight and responsive web application using React.js and Vite for fast development and performance. Users can add multiple items, adjust quantities, mark completed purchases, and keep track of their progress in real-time. The intuitive design makes it easy to use on both desktop and mobile devices.

The frontend development is focused on providing a clean and user-friendly experience, allowing users to manage their shopping lists effortlessly while keeping track of what has been bought and what is still pending.
